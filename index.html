HTML tags and Attributes

Html Tags

Tag Name Description
<h1-h6> Heading The element defines a heading It has a start tag and an end tag
<p> paragraph The element defines a paragraph It has a start tag and an end tag
<!DOCTYPE> DOCTYPE Defines the document type
<a> Hyperlink Defines a hyperlink
<body> Body Defines the document's body
<br> Break Defines a single line break
<button> Button Defines a clickable button
<b> Bold Defines bold text
<div> Div Defines a section in a document
<em> Emphasized Defines emphasized text
<embed> Embed Defines a container for an external application
<head> Head Contains metadata/information for the document
<img> Image Defines an image
<li> List Defines a list item
<main> Main Specifies the main content of a document
<meta> Meta Defines metadata about an HTML document
<nav> Navigation Defines navigation links
<section> Section Defines a section in a document
<html> HTML Defines the root of an HTML document
<span> Span Defines a section in a document
<strong> Strong Defines important text
<video> Video Defines embedded video content
<ol> Ordered List Defines an ordered list
<ul> Unordered List Defines an unordered list
<audio> Audio Defines embedded sound content
<input> Input Defines an input control

Html Attributes

Attribute Name Description
class Class Specifies one or more classnames for an element (refers to a class in a style sheet)
ID ID Specifies a unique id for an element
